## Prerequisites

Install _Unity 2020.3_
_Window_ -> _Package Manager_ -> add `ProBuilder`, remove `Visual Studio Code Editor`
Update `Visual Studio Editor`. Do not update `Rider`.
_Window_ -> _Asset store_ -> add `MicroSplat`
_Edit_ -> _Preferences_ -> _External tools_, set to `Visual Studio Code [...]`.
Regenerate project files.

Get plane and ring assets - `.fbx` and audio.

## Steps

### Get textures

https://freepbr.com/ for Unity textures

### Sculpt and paint the level

Create a terrain (1000x1000 by default).
Use the _Paint terrain_ -> _Raise or Lower_.
Create new layers.
Paint them.

### Add plane and movements

New script.

### DO not go through the terrain

Box colliders.
Rigidbody.
